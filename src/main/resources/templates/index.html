có 2 cách dùng log4j2 với springboot

cách 1: Log4j2 trực tiếp (nên dùng)

dùng Log4j2 trực tiếp như java core. Ko liên quan gì tới Springboot. Log4j2 ưu điểm hơn LogBack về performance

khi dùng với Devtool và LiveReload nó ko show log của Devtool và LiveReload khi restart app. log nay nhin rất rối mắt

https://howtodoinjava.com/log4j2/log4j-2-xml-configuration-example/
https://github.com/hungnguyenmanh82/JavaCore_testLog4j2

cách 2: gián tiếp qua Springboot (ko dùng)

cách này ko dùng đc version mới nhất của Log4j2 vì phụ thuộc vào Springboot. Ko jump code từ mành hình ConSole đc như cách 1 (đã test)

dùng Springboot dùng SLF4J API mặc định dùng Logback by default. Da đc ad vao mặc định web ap. Nếu dùng Logback thì phải học thêm Logback config mất thời gian

https://www.baeldung.com/spring-boot-logging
